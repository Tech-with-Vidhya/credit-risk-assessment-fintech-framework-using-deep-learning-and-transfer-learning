# Steps to Execute the Web Application

1. Download the project to your local directory
2. Open the project base folder from a terminal
3. Type "python3 app.py" and hit enter
4. Copy-paste the web url displayed in the terminal screen into a web browser. (Chrome Browser Recommended)
5. Credit Risk Assessment Calculator Web Aplication will get launched successfully
6. Input the responses to the credot risk factors related questions displayed on the application and click the "Calculate" button.
7. User will be re-directed to the results page where the calculated credit score in the three digit format and the probability of default in percentage will be displayed as the results.
